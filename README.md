# TripAI - תכנון טיולים חכם

אפליקציית תכנון טיולים מבוססת AI שמתאימה את עצמה לסגנון הטיול שלכם.

## פיצ'רים

- שאלון פרופיל מטייל חכם
- תכנון אוטומטי עם תעדוף (חובה / כדאי / אם יש זמן)
- המלצות אוכל לכל ארוחה
- תכנון העברות חכם
- צ'אט AI דינמי לשאלות תוך כדי טיול
- תמיכה ב-Google Gemini AI
- עברית RTL + Mobile First

## התקנה

```bash
npm install
npm run dev
```

## הגדרות AI

ניתן להשתמש במצב דמו (ברירת מחדל) או לחבר Google Gemini API key דרך דף ההגדרות.
